Flex Box

Intro to flex box

Flex Box stands for flexible box layout. It's a CSS 3 layout model that allows us to create complex layouts.

Once you set a display to flex on a container, children will become items. Those items will be displayed side-by-side.

The flex box layout is bidirectional, meaning you can arrange your elements either in rows or columns. Flex Box also allows us to create a responsive website.

Main and cross axis: All flex container elements containing a value of flex or inline-flex on display have both the main and cross axis. The default direction is row.

Flex direction

It is the flow of your flex items. In what direction are they aligned? Is it the x-axis or the y-axis?
